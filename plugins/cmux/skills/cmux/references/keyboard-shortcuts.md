# User Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New workspace |
| `Cmd+T` | New tab |
| `Cmd+P` | Go to workspace |
| `Cmd+D` | Split right |
| `Cmd+Shift+D` | Split down |
| `Cmd+Shift+P` | Command palette |
| `Cmd+Shift+R` | Rename workspace |
| `Cmd+Shift+L` | New browser |
| `Cmd+Shift+U` | Jump to latest unread |
